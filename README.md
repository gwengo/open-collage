# open-collage
A free, open collection of high-resolution collage assets — engravings, natural history illustration, vintage photography, watercolor, medieval manuscript imagery, and printed ephemera — made to be used.

Everything here is transparent PNG, ready to layer. Credit is appreciated, but not required.

---

## What's here

Images from any freely licensed source: museum collections, scanned books, public domain archives, original contributions. The only criteria are that the asset is freely licensed, high resolution, and useful to people making things.

Assets are organized by style and source type — the way you actually browse when you're mid-collage:

```
engraving/        — woodcut, etching, lithograph, pen illustration, crosshatch
natural-history/  — scientific specimen plates, botanical, anatomy, geology
photograph/       — halftone, photographic reproduction, press photo, portrait
watercolor/       — painted illustration, wash, hand-colored plates
medieval/         — manuscript, illuminated, devotional, bestiary, early woodcut
vernacular/       — ephemera, stamps, labels, packaging, trade cards
```

Within each folder, files are named `style-subject-number.png` — e.g. `engraving-vehicle-01.png`, `natural-history-primate-03.png`, `medieval-monster-02.png`.

---

## Curated and open

The archive has two tiers:

**`curated/`** — reviewed assets. Looked at closely for quality, usefulness, and consistency. The canonical collection.

**`open/`** — community submissions that meet the technical spec (resolution, format, naming) and are available to use immediately. Not yet deeply reviewed.

Both are usable. The distinction is about curation labor, not quality judgment. Both tiers use the same folder structure.

## Use

Download individual files or clone the whole repository.

```bash
git clone https://github.com/gwengo/open-collage.git
```

Or grab raw URLs directly from GitHub for use in Are.na, p5.js, or anywhere that accepts image links.

Good for: digital collage, zine layout, risograph layer separation, generative art, textile design, screen printing positives, projection, print-on-demand.

## Contribute

Read CONTRIBUTING.md before submitting.

The short version: 600dpi minimum, transparent background, PNG, named to convention, placed in the right folder under `open/`. Open a pull request with a short note about what you're adding and where it came from.

This archive is collective by design. If you have images that belong here, they belong here.
