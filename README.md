# open-collage
A free, open collection of high-resolution collage assets — textures, marks, edges, grounds, type fragments, photographic material — made to be used for digital collages and zines. Credit is appreciated, but not required.

## What's here

Images from source like scanned print, photograph, found material, hand-drawn, digitally processed. The only criteria are that the asset is freely licensed, high resolution, and useful to people making things. Assets are organized by the following:

```
texture/     — surface, grain, noise, material
edge/        — borders, cuts, torn edges, dividing lines
mark/        — discrete gestures, stamps, single strokes
ground/      — large areas, fields, washes, backgrounds
figure/      — isolated shapes with clear silhouette
void/        — negative space, open areas, absence
pattern/     — repeating elements, grids, tessellation
type/        — letterforms, hand-lettered, printed text fragments
photo/       — photographic fragments, halftones, reproductions
```

Within each folder, files are named `series-quality-number.png` — e.g. `river-edge-01.png`, `studio-texture-03.png`.

## Use

Download individual files or clone the whole repository.

```bash
git clone https://github.com/gwengo/open-image-marks.git
```

Or grab raw URLs directly from GitHub for use in Are.na, p5.js, or anywhere that accepts image links.

Good for: digital collage, zine layout, risograph layer separation, generative art, textile design, screen printing positives, projection, print-on-demand.

## Contribute

Read [CONTRIBUTING.md] before submitting.

The short version: 600dpi minimum, transparent background, PNG, named to convention, placed in the right folder. Open a pull request with a short note about what you're adding and where it came from. Submissions go into `open/` first.

This archive is collective by design. If you make images — scanned, photographed, drawn, processed — and want them here, they belong here.
